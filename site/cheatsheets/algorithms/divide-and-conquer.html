Divide and Conquer

Master Theorem

Given any recurrence of the form $T(n) = a T(\frac{n}{b}) + c n^k$ for all $n > b$, we have:

  • If $a > b^k$, then $T(n) = \Theta(n^{\log_b a})$
  • If $a < b^k$, then $T(n) = \Theta(n^k)$
  • If $a = b^k$, then $T(n) = \Theta(n^k \log n)$
